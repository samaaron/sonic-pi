# Get current arg checking status

```
current_arg_checks 
 <!--- #tr --><!--- #end tr -->
```


Returns the current arg checking setting (`true` or `false`).

This can be set via the fns `use_arg_checks` and `with_arg_checks`.

Introduced in v2.0
