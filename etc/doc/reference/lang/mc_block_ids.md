# Minecraft Pi - list all block ids

```
mc_block_ids 
 <!--- #tr --><!--- #end tr -->
```


Returns a list of all the valid block ids as numbers. Note not all numbers are valid block ids. For example, 19 is not a valid block id.

Introduced in v2.5
