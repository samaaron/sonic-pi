# Get current tempo

```
current_bpm 
 <!--- #tr --><!--- #end tr -->
```


Returns the current tempo as a bpm value.

This can be set via the fns `use_bpm`, `with_bpm`, `use_sample_bpm` and `with_sample_bpm`.

Introduced in v2.0
